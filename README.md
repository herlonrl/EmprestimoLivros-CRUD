# Empréstimo de Livros
CRUD básico.
Projeto de cadastro de livros para empréstimo simples com controle de dados em BD.

# Tecnologias e ferramentas utilizadas
* Visual Studio
* ASP.NET Core MVC
* SQLServer
* Bootstrap
* ClosedXML

# Funcionalidades
Cadastro, Busca, Atualização, Exclusão e Exportação dos dados.
